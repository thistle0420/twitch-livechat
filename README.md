# twitchLiveChat取得

## 概要

twitchのアーカイブからチャット一覧を取得する。

投稿日時、ユーザ名、コメントを抽出。

取得結果はGoogleSpreadsheetに出力する。

![image](https://github.com/thistle0420/twitch-livechat/assets/85113388/69877a71-dc83-46d6-bd09-f540f62dad08)

時間ごとのコメント数から盛り上がり度の分析も可能
